# 阿里云机器翻译-通用翻译接口调用说明

注意事项

1、QPS限制50，如有扩展需求，请与我们联系。钉钉群号：23369411

2、字符长度上限是5000字符，若超出上限，可以保存成文件，使用[文档翻译](https://help.aliyun.com/zh/machine-translation/product-overview/introduction-to-document-translation)

3、请确保在使用该接口前，已充分了解通用版翻译产品的收费方式和价格

## 调试

[您可以在OpenAPI Explorer中直接运行该接口，免去您计算签名的困扰。运行成功后，OpenAPI Explorer可以自动生成SDK代码示例。](https://api.aliyun.com/#product=alimt&api=TranslateGeneral&type=RPC&version=2018-10-12)

## 请求参数

| 名称  | 类型  | 是否必选 | 示例值 | 描述  |
| --- | --- | --- | --- | --- |
| Action | String | 是   | TranslateGeneral | 系统规定参数。取值：TranslateGeneral。 |
| FormatType | String | 是   | text | 翻译文本的格式，html（ 网页格式。设置此参数将对待翻译文本以及翻译后文本按照html格式进行处理）、text（文本格式。设置此参数将对传入待翻译文本以及翻译后结果不做文本格式处理，统一按纯文本格式处理。 |
| Scene | String | 是   | general | 通用版本默认是：general |
| SourceLanguage | String | 是   | zh  | 原文语言[参考语言code](https://help.aliyun.com/zh/machine-translation/product-overview/general-version-of-machine-translation) |
| SourceText | String | 是   | 你好  | 需要翻译的内容 |
| TargetLanguage | String | 是   | en  | 译文语言[参考语言code](https://help.aliyun.com/zh/machine-translation/product-overview/general-version-of-machine-translation) |
| Context | String | 否   | 早上我在家里吃了面包 | 上下文信息，可选填 |

## 返回数据

| 名称  | 类型  | 示例值 | 描述  |
| --- | --- | --- | --- |
| Code | Integer | 200 | 错误码 |
| Data | Struct |     | 返回内容 |
| Translated | String | Hello | 翻译结果 |
| WordCount | String | 10  | 总单词数 |
| DetectedLanguage | String | zh  | zh源语言传入auto时，语种识别后的源语言代码 |
| Message | String | success | 返回信息 |
| RequestId | String | 86D18195-D89C-4C8C-9DC4-5FCE789CE6D5 | 请求ID |

## 示例

请求示例

```
http(s)://[Endpoint]/?Action=TranslateGeneral
&FormatType=text
&Scene=general
&SourceLanguage=zh
&SourceText=你好
&TargetLanguage=en
&<公共请求参数>
```

公共请求参数参见[《RPC 调用机制》公共请求参数组装方案](https://help.aliyun.com/zh/sdk/product-overview/rpc-mechanism)

正常返回示例

`XML` 格式

```
<TranslateGeneralResponse>
      <RequestId>14E447CA-B93B-4526-ACD7-42AE13CC2AF6</RequestId>
      <Data>
            <Translated>Hello</Translated>
      </Data>
      <Code>200</Code>
</TranslateGeneralResponse>
```

`JSON` 格式

```
{
    "TranslateGeneralResponse": {
        "RequestId": "14E447CA-B93B-4526-ACD7-42AE13CC2AF6",
        "Data": {
            "Translated": "Hello"
        },
        "Code": 200
    }
}
```
